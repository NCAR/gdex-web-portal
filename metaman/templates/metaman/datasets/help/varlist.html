Detailed Variables

You can specify detailed variable names and their units, and you can point to a document that contains a list of detailed variables.

If you include a list of detailed variables, it will be indexed for the dataset search. Enter one variable name per line. You can optionally also specify the units for the variable on the same line. The units must be separated from the variable name with the "::" (double colon) string. Whitespace is allowed in the variable names and units and will not be used as a separator.

There is no standard format for the units, but if you use the "^" (carat) to indicate an exponent, it can be transformed for a nicer appearance in HTML. Example - specify the units as m s^-1 and it can be made to look like m s-1 on a web page.

To point to a document containing detailed variable information, enter the full URL on one line, beginning with "http:", "https:", or "ftp:". If you enter more than one URL, only the first one will be accepted. The information in the document you point to can be in any format you choose. It will not be indexed for the dataset search. Therefore, this is a sub-optimal way to specify detailed variables.
